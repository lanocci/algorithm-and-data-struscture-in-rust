## Queue

- 「待ち行列」
- データを到着順に処理
- FIFO
- 主なメソッド
  - enqueue
  - dequeue
- 待ち行列にはリングバッファを利用するのが効率的
  - イメージとしては環状の配列
  - 普通の配列だと、要素を出し入れする度にインデックスがどんどん大きくなって行ってしまうので、使っているうちに配列の頭の方が空っぽになってしまう
  - リングバッファはenqueue / dequeue するときに head と tailの位置をずらすことで効率的に容量を使うことができる

### Rustでは

- `std::collections::VecDeque` がリングバッファを実装している
